Overview

The SPARQL ascending sort operator.

[24] OrderCondition ::= ( ( ‘ASC’ | ‘DESC’ ) BrackettedExpression ) | ( Constraint | Var )

Examples:

SPARQL Query

PREFIX foaf:    <http://xmlns.com/foaf/0.1/>
SELECT ?name
WHERE { ?x foaf:name ?name }
ORDER BY ASC(?name)

SSE

(prefix ((foaf: <http://xmlns.com/foaf/0.1/>))
  (project (?name)
    (order ((asc ?name))
      (bgp (triple ?x foaf:name ?name)))))
